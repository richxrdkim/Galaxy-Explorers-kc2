Galaxy Explorers Website

This is a simple space-themed website built using HTML, CSS, and Bootstrap.

Features:
- Navigation bar with Home, About, and Contact links
- Registration form with fields for first name, last name, email, and password
- Responsive table showing example user data
- Large banner image and circular profile image
- Two buttons with responsive display
- Mobile-friendly layout using Bootstrap

File Info:
- index.html is the main file (in the root folder)
- /images contains the image files
- /css (optional) can hold custom styles

How to Use:
Open index.html in your web browser. Make sure the images folder is in the right place so the pictures load.

This project was made for practice using Bootstrap and basic web design.
